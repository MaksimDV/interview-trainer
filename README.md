# Interview Trainer MVP

## Project Overview
Interview Trainer is a flashcard-based system for practicing interview questions.  
It allows users to answer multiple-choice questions, get scored, and track progress.

## Features (MVP)
- Random questions by topic
- Multiple-choice answers
- Scoring system (0–10)
- Progress tracking

## Tech Stack
- **Backend:** Java + Spring Boot
- **Database:** PostgreSQL
- **Frontend:** minimal (React or Thymeleaf)
- **Documentation:** Markdown files in `docs/`
